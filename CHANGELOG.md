# Changelog

## 0.1.0

- Initial public release of `pi-extension-search`
- Adds `exa_search`
- Adds `brave_search`
- Supports API keys from shell env or Pi `settings.json`
- Adds automated tests for settings loading and provider adapters
- Adds CI validation for tests, bundle health, and package contents
- Adds release workflow for tagged GitHub releases
- Adds contribution and release documentation
